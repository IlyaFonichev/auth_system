#  Система аутентификации и авторизации

Полнофункциональная система аутентификации и авторизации с кастомной моделью пользователя, системой ролей и разрешений, построенная на Django и Django REST Framework.

##  Основные возможности

###  Аутентификация
- **Кастомная модель пользователя** на основе email
- **JWT аутентификация** для API
- **Сессионная аутентификация** для веб-интерфейса
- **Единая точка входа** с автоматическим перенаправлением
- **Безопасная регистрация** с подтверждением пароля

###  Управление пользователями
- **Регистрация новых пользователей** через веб-форму
- **Автоматическое определение прав** при входе
- **Редактирование профиля** (имя, фамилия, отчество)
- **Мягкое удаление аккаунтов** (is_active=False)
- **Выход из системы**

###  Система авторизации
- **Гибкая система ролей и разрешений**
- **Разграничение доступа к ресурсам**
- **Автоматические проверки прав** для API endpoints
- **Административный интерфейс** для управления правами

###  Автоматическое перенаправление
- **Обычные пользователи** → Личный кабинет (`/profile/`)
- **Администраторы/Суперпользователи** → Админка Django (`/admin/`)

##  Технологии

- **Backend**: Django 4.2, Django REST Framework
- **База данных**: PostgreSQL
- **Аутентификация**: JWT + Session Auth
- **Frontend**: HTML5, CSS3, JavaScript
- **Стили**: Modern CSS
